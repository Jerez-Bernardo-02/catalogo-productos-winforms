# Catálogo de Productos

## Descripción
Aplicación de escritorio para la gestión de artículos de un catálogo comercial.
Permite administrar productos, marcas y categorías, realizar búsquedas,
altas, bajas y modificaciones, y visualizar detalles.

## Funcionalidades
- Listado de artículos
- Búsqueda por distintos criterios
- Alta, baja y modificación de artículos
- Gestión de marcas y categorías
- Visualización de detalles del producto

## Tecnologías utilizadas
- C# / .NET (WinForms)
- SQL Server
- Arquitectura en capas

## Base de datos
El script de creación de la base de datos se encuentra en la carpeta `/db`.

## Observaciones
Proyecto desarrollado como trabajo práctico académico.
